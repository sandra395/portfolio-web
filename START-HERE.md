## ğŸ†˜ Need Help?

### If Something Breaks:
1. **Check the Terminal** - It shows error messages
2. **Undo your last change** - Cmd+Z (Mac) or Ctrl+Z (Windows)
3. **Look for common mistakes:**
   - Missing comma `,`
   - Missing quote `"`
   - Wrong file path
4. **Read the error message** - It often tells you what's wrong

### Common Errors:

**"Cannot find module"**
- You might have a typo in a file path
- Check image paths start with `/images/`

**"Unexpected token"**
- You're missing a comma `,` or quote `"`
- Check your last change

**"Module parse failed"**
- You might have broken the syntax
- Undo your last change and try again
